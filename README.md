# Система управления сотрудниками и подразделениями

**Корпоративная система учёта сотрудников с иерархией подразделений**

## Описание проекта

Проект представляет собой веб‑приложение для учёта сотрудников компании с поддержкой иерархической структуры подразделений. Система позволяет:
* управлять подразделениями (создание, редактирование, удаление);
* вести учёт сотрудников;
* отслеживать принадлежность сотрудников к подразделениям;
* осуществлять поиск и фильтрацию данных.

## Технологии

* **Язык программирования:** Go (Golang) 1.20+
* **Веб‑фреймворк:** Gin
* **ORM:** GORM (с поддержкой PostgreSQL/MySQL)
* **База данных:** PostgreSQL (рекомендуется) или MySQL
* **Логирование:** кастомный логгер с записью в файл и консоль
* **Структура:** Clean Architecture (разделение на слои: handlers, services, repository, models)

## Функциональные возможности

### Управление подразделениями
* создание новых подразделений;
* редактирование информации о подразделениях;
* удаление подразделений (с проверкой наличия сотрудников);
* просмотр иерархии подразделений (родительские/дочерние);
* получение списка всех подразделений.

### Управление сотрудниками
* добавление новых сотрудников;
* редактирование данных сотрудников;
* удаление сотрудников;
* назначение сотрудников в подразделения;
* просмотр информации о сотрудниках с данными о подразделении;
* поиск сотрудников по имени или должности.
